# High-Converting Landing Page

A **fast-loading, fully responsive landing page** designed to boost conversions, capture leads, and create a professional online presence. Perfect for portfolios, paid ad campaigns, and lead generation funnels.

---

## 🔹 Features

- **Hero Section:** Engaging headline, subtext, and call-to-action button.  
- **Services Section:** Clear presentation of services with interactive cards.  
- **Quote Form:** Simple and user-friendly form to capture visitor information.  
- **Call-To-Action Section:** Encourages visitors to take the next step.  
- **Ultra-Lightweight & Fast:** Minimal HTML, CSS, and JavaScript for instant loading.  
- **Mobile Responsive:** Works seamlessly on mobile, tablet, and desktop.  
- **Subtle Animations:** CSS-only fade-in animations for a modern, interactive feel.  

---

## 🔹 Tech Stack

- HTML5  
- CSS3 (Grid & Flexbox)  
- Minimal JavaScript (for form submission alert)  
- Google Fonts (Inter)
